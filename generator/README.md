# Site generator

`index.html` is generated by inlining the wrangler logic, a
dependency-free ZIP writer, the UI, and the entire `site-template/` into one
file. 

Sources:
- `generator/template.html` — UI shell + styles (with inject placeholders)
- `generator/src/generator-ui.js` — two-stage UI logic
- `generator/src/zip.js` — minimal STORE-method ZIP writer (no deps)
- `generator/src/site-single.js` — from-scratch single-page site builder
  (vanilla HTML+JS, ported CSS, JSON-embedded data)
- `../wrangler/wrangler.js` — Swiss-papers extraction (shared with Node)
- `../site-template/**` — the React site packaged into the ZIP output



## Rebuild the generator
After changing any of those sources, rebuild:

```bash
node generator/build.mjs
```


## Testing the generator

`node generator/test/single-site.test.mjs` (Liquid-safety, JSON
round-trip, script-tag integrity; writes `generator/test-single.html` for a
manual look).
