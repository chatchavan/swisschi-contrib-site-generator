🇨🇭 SwissCHI Site Generator

Turn a SIGCHI program JSON into a deployable Swiss-contributions site. The two stages are independent.

① Extract papers → CSV

Drop a SIGCHI program JSON (the “Get Conference data JSON” download). You get a swiss_papers.csv to curate manually or with collaborators (awards, fixes, extra papers, messages).

Drop program JSON here
or click to choose a file

② (Optional) Replacing program URL with DOI

After the conference, instead of linking to the program page, you may want to link directly to the DOI link of each paper. This step needs to be done on a separate Python app that will query DOI of each paper in the CSV. See the app in the folder doi-resolver.

③ Build deployable site ← curated CSV

Drop your (curated) CSV and fill the conference details. You get a ZIP of a deployable site with the CSV embedded — unzip, then npm install && npm run deploy.
